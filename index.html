Module Details CFAMS + CFAMO
This system provides Fixed Asset Management Setup (CFAMS) and operations (CFAMO) covering the full asset lifecycle: Request → Approval → Procurement → Transfer → Disposal → Repair → Revaluation → Surrender.

All operations follow maker-checker (4-eyes) principle — a record created by one user must be verified by a different user before it takes effect.
